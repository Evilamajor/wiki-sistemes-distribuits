# Notes

Recull de resums i apunts organitzats per mòdul del curs de Sistemes Distribuïts. Cada carpeta conté els fitxers Markdown corresponents a cada tema.

## Índex de carpetes
- [`00_introduccio/`](./00_introduccio/): definició i propietats bàsiques dels sistemes distribuïts.
- [`01_comunicacio/`](./01_comunicacio/): models de comunicació, middleware i acoblament temporal/espacial.
- [`02_consistencia/`](./02_consistencia/): consistència forta i eventual, rellotges de Lamport i de vector.
- [`03_TSAE/`](./03_TSAE/): apunts sobre el protocol TSAE (fases, purga i experiments).
- [`04_models/`](./04_models/): models i arquitectures (Chord, consistent hashing, TrueTime/Spanner, blockchain mínima).
- [`05_aplicacions/`](./05_aplicacions/): aplicacions i plataformes (Kafka, RabbitMQ, Cassandra, DynamoDB).
- [`06_referencia/`](./06_referencia/): referències pràctiques (estructura del repo, ordres git, glossari, exemples Mermaid).

## Afegir nous apunts
1. Crea un fitxer Markdown dins la carpeta adequada.
2. Usa noms descriptius en minúscules separats per guions baixos (p. ex. `nou_tema.md`).
3. Si cal, actualitza aquesta llista perquè l’índex es mantingui al dia.
